Vault allows you to keep sensitive data (passwords/keys/etc.) in encrypted files


You can also set ANSIBLE_VAULT_PASSWORD_FILE environment variable, e.g. ANSIBLE_VAULT_PASSWORD_FILE=~/.vault_pass.txt and Ansible will automatically search for the password in that file.

From <https://docs.ansible.com/ansible/latest/user_guide/playbooks_vault.html> 






https://docs.ansible.com/ansible/latest/user_guide/playbooks_vault.html

Setting up vaulted file

usage: ansible-vault [-h] [--version] [-v]
                  {create,decrypt,edit,view,encrypt,encrypt_string,rekey}
                  ...

Running a playbook w/ Vault

ansible-playbook site.yml --ask-vault-pass
